## Run Locally

Clone the project

```bash
  git clone https://github.com/hackkosice/hk-advent
```

Go to the project directory

```bash
  cd hk-advent
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```


## Deployment

GitHub Actions are set up to deploy after pushing to `main` and PR

To deploy this project manually, run

```bash
  firebase login
  npm run build
  firebase deploy
```
